About AlbumBrowser

A modern Angular SPA built for Web Development Lab 6

📖 About the App

AlbumBrowser is a multi-page Single Page Application built with Angular 17+. It demonstrates core Angular concepts including routing, HTTP services, Observables, CRUD operations, and reactive data handling using the async pipe.

🛠️ Technologies Used

  • Angular 17+ (Standalone Components)
  • Angular Router (with route parameters)
  • HttpClient & RxJS Observables
  • JSONPlaceholder REST API
  • TypeScript (strict interfaces)
  • Component-scoped CSS
